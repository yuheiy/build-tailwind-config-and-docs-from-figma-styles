# Styles

sour-reactでは、[Tailwind CSS](https://tailwindcss.com/)のテーマをカスタマイズして使用しています。また、Tailwind CSSの組み込みのユーティリティに加えて、sour-react独自のコンポーネントやユーティリティも併せて提供しています。この項では、これらsour-react独自のスタイルについて説明します。

## colors

colorsは[Figmaファイル](https://www.figma.com/design/y4Dqe1SvQrU0zFfmanab8v/Sour-for-KARTE)のcolor stylesをもとにして定義されています。color stylesのうち、Semantic colorsはTailwind CSSの一部のユーティリティでのみ、Primitive colorsは[色を扱うすべてのユーティリティ](https://tailwindcss.com/docs/colors#using-color-utilities)で使用できます。

### text-color

color stylesの`Semantic/Text/*`は、[colorユーティリティ](https://tailwindcss.com/docs/color)で使用できます。

<TextColor />

### background-color

color stylesの`Semantic/Bg/*`は、[background-colorユーティリティ](https://tailwindcss.com/docs/background-color)で使用できます。

<BackgroundColor />

### border-color

color stylesの`Semantic/Border/*`は、[border-colorユーティリティ](https://tailwindcss.com/docs/border-color)で使用できます。

<BorderColor />
<DivideColor />

### Primitive colors

color stylesの`Primitive/*`は、[色を扱うすべてのユーティリティ](https://tailwindcss.com/docs/colors#using-color-utilities)で使用できます。`text-primitive-green-600`のようなクラスから使用できます。

基本的には、Primitive colorsではなくSemantic colorsを使用してください。Semantic colorsが使用できない場合のみ、Primitive colorsを使用してください。

## font-size

Sourにおいて一般的に使用される`font-size`と`line-height`の組み合わせが[font-sizeユーティリティ](https://tailwindcss.com/docs/font-size)として定義されています。

基本的には、font-sizeユーティリティではなくtypographyを使用してください。typographyが使用できない場合のみ、font-sizeユーティリティを使用してください。
